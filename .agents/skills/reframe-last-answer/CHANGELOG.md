# Changelog

All notable changes to this skill are documented here.

## [0.1.0] - 2026-06-01

### Added
- Initial release. Reframes the assistant's own previous detailed answer into a decision-first brief: verdict first, plain language, lossless on anything that changes a decision, lossy on process, detail offloaded to a pointer.
- Adaptive format — picks the shape that fits the answer (decision, comparison, research, risk) rather than a fixed template.
- Dual invocation: slash command and auto-invoke on phrases like "reframe your last answer" or "the executive version".
